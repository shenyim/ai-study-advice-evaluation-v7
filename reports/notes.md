# Project Notes

## Current Product Level

This version is no longer just a basic dashboard. It is a productized MVP that demonstrates the workflow of an internal AI quality evaluation system.

## Best Portfolio Angle

Use this angle:

> I designed and built a human-centered AI evaluation platform for student-facing advice systems. The platform compares multiple AI model outputs, applies risk-aware scoring, and flags cases that require human review.

## What Makes It HCAI-DS

- Human-centered metrics, not only technical metrics.
- Risk-aware evaluation.
- Student decision-making context.
- Model comparison.
- Failure taxonomy.
- Dashboard for product and research interpretation.

## What To Improve Next

The next highest-impact improvement is connecting real model outputs. After that, add human raters and qualitative coding.
