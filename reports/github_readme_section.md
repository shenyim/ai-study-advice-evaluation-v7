# GitHub README Section

## Portfolio Positioning

This project is a human-centered AI evaluation platform for student-advice systems. It demonstrates how to move beyond chatbot demos by evaluating answer quality, risk control, human annotation reliability, statistical evidence, and product readiness.

It is suitable for discussing AI evaluation, responsible AI, product data science, HCI research operations, and model-comparison methodology. It is a portfolio and research demo, not a deployed student-advice product.
