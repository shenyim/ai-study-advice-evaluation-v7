# 3-Minute Demo Script

## 0:00–0:30 — Problem

Student-facing AI assistants can give polished but risky advice. Normal accuracy metrics are not enough because academic and career advice involves uncertainty, personalization, and possible harm.

## 0:30–1:10 — Product

This project is a human-centered AI evaluation platform. It compares AI-generated student advice across accuracy, helpfulness, clarity, actionability, and risk control. It also uses risk-weighted scoring so high-risk mistakes count more.

## 1:10–1:50 — Dashboard walkthrough

In the overview tab, I check product health. In model comparison, I compare candidate models by weighted score and review rate. In case review, I inspect individual failures and translate them into product fixes.

## 1:50–2:30 — HCAI-DS value

The key idea is not just building a chatbot. The key idea is building a quality and governance layer around AI advice: rubric design, risk-sensitive evaluation, review queues, and product guardrails.

## 2:30–3:00 — Next step

The current version uses simulated outputs to demonstrate the workflow. The next milestone is to plug in real outputs from multiple LLMs and run human annotation using the included annotation guide.
